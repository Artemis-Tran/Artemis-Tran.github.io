    Projects

    Tanks

    Tanks! is a recreation of the classic Wii Play "Tanks!" game built using purely JavaScript and the Tiny Graphics library. This game features a user controlled a tank via mouse and keyboard to avoid enemy fire, shooting bullets that ricochet off walls, and placing bombs which explode after a duration or can be exploded by a bullet. The goal of the game is to eliminate all enemy tanks without getting hit before the level timer ends, and advance through all possible levels to win the game!

    PlantDex

    PlantDex is an Android app built with React Native that encourages user to go outside and take pictures of plants! Currently, users are able to snap photos of plants they come across which is saved into their gallery. Other features on the way is the ability to add friends, store notes about each plant, and add music.
